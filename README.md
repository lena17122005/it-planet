# Трамплин

Полноценный MVP карьерной платформы для взаимодействия соискателей, работодателей и кураторов.

## Реализовано

### Backend (FastAPI)
- JWT access + refresh flow (`/auth/login`, `/auth/refresh`, `/auth/register`, `/auth/verify-email`).
- Ролевая модель и защищенные эндпоинты (`seeker`, `employer`, `curator`).
- CRUD-потоки для вакансий, откликов, контактов, модерации и верификации компаний.
- Геокодирование адресов через Nominatim.
- Автоматическое создание администратора-куратора при старте БД.
- Миграция Alembic с таблицами для пользователей, компаний, вакансий, откликов, контактов, тегов и логов модерации.

### Frontend (React + TypeScript)
- Обновленный интерфейс: современная карточная вёрстка, sticky header, glass-стиль.
- Главная: карта Leaflet + фильтры + лента вакансий + модальное окно карточки.
- Цветные маркеры: серый (обычно), синий (избранное), зеленый (мероприятие).
- Избранное неавторизованных в `localStorage`.
- Страницы входа/регистрации с выбором роли и дополнительными полями для работодателя.
- JWT refresh на фронтенде через Axios interceptor.
- Role-based protected routes для кабинетов.

### Тестирование
- Backend E2E-тест `backend/tests/test_e2e_flows.py` покрывает:
  - регистрацию ролей,
  - verify email,
  - refresh token,
  - создание вакансии работодателем,
  - публикацию куратором,
  - отклик соискателя,
  - сценарий контактов.

## Запуск
1. `cp .env.example .env`
2. `docker-compose up --build`
3. Открыть:
   - Frontend: http://localhost:3000
   - API Docs: http://localhost:8000/docs
